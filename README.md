# Mini-projet Web : Site d'information sur la guerre en Iran (inspiré du site d'actualité LeMonde ) 

## Description du projet
Ce projet est un mini-site d'information sur la guerre en Iran, inspiré du site d'actualité LeMonde. Il a été développé dans le cadre d'un projet de développement web. Le site présente des articles, des images et des vidéos liés à la guerre en Iran, offrant aux utilisateurs une source d'information complète et accessible.

## Technologies utilisées
- HTML5 : pour la structure du site
- CSS3 : pour la mise en page et le design du site
- JavaScript : pour les interactions et les fonctionnalités dynamiques du site
- PHP : pour la gestion des données et l'affichage dynamique du contenu
- PostgreSQL : pour la gestion de la base de données contenant les articles, les images et les vidéos
- Docker : pour la conteneurisation de l'application et faciliter le déploiement

## Fonctionnalités

### Front-Office : 
- Affichage des articles avec images et vidéos
    - Page d'accueil présentant les derniers articles
    - Page de détail pour chaque article avec contenu complet, images et vidéos associées
    - Catégories d'articles (politique, économie, culture, etc.) 
- Barre de recherche pour trouver des articles spécifiques
    - Filtrage des articles par mots-clés ou catégories
    - Affichage des résultats de recherche avec liens vers les articles correspondants

### Back-Office :
- Interface d'administration pour la gestion du contenu
    - Formulaire de création et de modification d'articles avec champs pour le titre, le contenu, les images et les vidéos
    - Liste des articles existants avec options de modification et de suppression
    - Gestion des catégories d'articles
- Système d'authentification pour sécuriser l'accès à l'administration
